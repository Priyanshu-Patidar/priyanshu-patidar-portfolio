## ✨ Features

### 💎 Premium Experience
- **Glassmorphism UI**: Modern, translucent design system built with Tailwind CSS.
- **Micro-interactions**: Subtle, performant animations using **Framer Motion** and **GSAP**.
- **Performance Optimized**: Lighthouse scores target 95+ via route-based code splitting and non-blocking asset loading.
- **Custom Cursor & Progress**: Interactive custom cursor and scroll progress bars for enhanced engagement.

### 🧠 Strategic Project Showcases
- **SmartPark (Enterprise SaaS)**: Full-stack parking management with STOMP WebSockets and Dynamic Surge Pricing.
- **AI Resume Analyzer (RAG)**: Sophisticated LLM pipeline using LangChain, LangGraph, and Supabase pgvector.
- **Interactive Case Studies**: 12-section deep dives into architecture, database design, and performance optimizations.

### 📊 Professional Tooling
- **Vercel Analytics**: Real-time production tracking.
- **SEO Ready**: JSON-LD Person schema, OpenGraph tags, and Twitter Cards for maximum visibility.
- **Recruiter Optimized**: Command Palette for quick navigation and one-click resume access.

## 🏗️ Project Structure

```text
/src
  /components
    /layout      # Shared UI (Navbar, Footer, Progress)
    /sections    # Main landing page sections
    /ui          # Reusable premium components (Modals, Timelines)
  /data          # Content-driven architecture (Projects, Case Studies)
  /hooks         # Custom logic (Analytics, Theme, Intersection Observers)
  /types         # TypeScript interfaces
  /utils         # Helper functions
/public          # Static assets (Images, SEO, Icons)
```

## 🚦 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/Priyanshu-Patidar/priyanshu-patidar-portfolio.git
cd priyanshu-patidar-portfolio
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run development server
```bash
npm run dev
```

### 4. Build for production
```bash
npm run build
```
